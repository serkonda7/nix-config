# NixOS Config
```sh
sudo loadkeys de
sudo -i

curl -LO github.com/serkonda7/nixos-config/archive/main.zip
unzip main.zip

cd nixos-config-main
./install.sh
# On reboot press F12 and select the hard drive
```
