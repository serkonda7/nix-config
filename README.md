# nix-conig
```sh
sudo loadkeys de

curl -LO github.com/serkonda7/nix-config/archive/main.zip
unzip main.zip

cd nix-config-main
sudo ./install.sh
# On reboot press F12 and select the hard drive

# After login:
./first_run.sh
```
